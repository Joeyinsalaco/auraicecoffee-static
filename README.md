# AURA Iced Coffee Website

This is a Vite project built from the uploaded AURA JSON backup.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

Use these settings:

- Framework: Vite
- Build Command: npm run build
- Output Directory: dist
- Install Command: npm install

## Files

- `src/site-data.json` contains your original site content.
- `src/main.jsx` renders the site.
- `src/styles.css` controls the design.
- `public/assets` contains extracted uploaded images from the backup.
